# Personal Portfolio Website

A modern, responsive personal portfolio website built using **HTML5** and **CSS3**. This project was developed as part of a **Cloud Computing & DevOps** assignment and deployed using **Amazon S3** with **Amazon CloudFront** for secure HTTPS hosting.

- Responsive design for desktop, tablet, and mobile

## ☁️ Deployment Steps

1. Create an AWS S3 bucket.
2. Upload all project files.
3. Enable Static Website Hosting.
4. Configure the bucket policy for public access.
5. Create a CloudFront distribution using the S3 bucket as the origin.
6. Wait for deployment and access the website using the CloudFront HTTPS URL.
